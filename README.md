# Установка openwrt
Обновление берем от ZBD WA-05(https://openwrt.org/toh/zbt/wa-05).

Далее 2 варианта:
- заменяем EN25QH64 (пока ее нет в ядре, патч уже есть) на 25Q64.
- используем патч и перекомпилируем

Обновление возможно только через бутлоадер.
